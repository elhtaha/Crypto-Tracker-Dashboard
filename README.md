 Crypto Tracker Dashboard
A web-based crypto tracker that displays real-time cryptocurrency data using a public API.  
The application allows users to select a coin and view key market information such as price, 24-hour change, market capitalization, volume, and overall market trend.

Features
- Real-time cryptocurrency data
- Supports 50+ coins
- Clean dashboard UI with stat boxes
- Market trend indicator (Bullish / Bearish / Neutral)
- Color-coded feedback (green / red / yellow)
- Responsive design

Technologies Used
- HTML
- CSS
- JavaScript
- CoinGecko API

 How It Works
1. The user selects a cryptocurrency from the dropdown list.
2. The app sends a request to the CoinGecko API.
3. The API returns real-time market data.
4. The dashboard updates dynamically with:
   - Current price
   - 24h price change
   - Market cap
   - Trading volume
   - Market trend
